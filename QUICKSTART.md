# 快速启动指南

## 🚀 5分钟快速启动

### 1. 安装依赖

\`\`\`bash
cd frontend
npm install
\`\`\`

### 2. 启动后端服务

在另一个终端窗口：

\`\`\`bash
cd ..  # 回到项目根目录
python app.py
\`\`\`

后端应该运行在 `http://localhost:5000`

### 3. 启动前端开发服务器

\`\`\`bash
npm run dev
\`\`\`

### 4. 打开浏览器

访问 `http://localhost:3000`

🎉 现在您可以开始使用电厂智能问答系统了！

## 📝 使用说明

### 开始对话

1. 点击左上角"新对话"按钮创建会话
2. 在底部输入框输入问题
3. 按 `Ctrl/Cmd + Enter` 或点击"发送"按钮
4. 等待 AI 助手回答

### 会话管理

- **切换会话**: 点击左侧会话列表中的会话
- **删除会话**: 鼠标悬停在会话上，点击删除图标
- **查看历史**: 切换会话后自动加载历史消息

### 查看数据来源

每条回答下方会显示数据来源标签：
- 📚 **知识库** - 从文档检索的信息
- 🗄️ **数据库** - SQL 查询结果
- 🔌 **实时数据** - API 接口数据
- 💬 **对话** - LLM 直接回答

### 查看系统状态

点击右上角"系统状态"按钮查看：
- Ollama 服务运行状态
- 当前加载的模型
- 模型可用性

## 🔧 常见问题

### Q: 页面空白

**A:** 检查：
1. 后端是否运行（`http://localhost:5000/health`）
2. 浏览器控制台是否有错误
3. Node.js 版本是否 >= 18

### Q: 无法发送消息

**A:** 检查：
1. 后端服务是否正常
2. 网络面板中 API 请求是否成功
3. 是否创建了会话

### Q: 会话丢失

**A:** 会话保存在浏览器 LocalStorage 中：
1. 不要使用无痕/隐私模式
2. 不要清除浏览器数据
3. 同一浏览器、同一域名下数据共享
